Git Bash Commands

Some Useful Commands :

1)ctrl+l => to clear screen
for commiting 

1)use git init => it will initialize a git repository
2)use git add filename => it will add files to staging area
3)use git commit -m "/Put a message" => it will commit the files

use git add .  =>add all files to staging area
use git status =>show the status of the files 
use git log => shows the commited files with message

final git command => git push -u origin master
for creating a new branch => git branch "branch name"
for switching to a new branch => git checkout "branch name"

git pull 
Pulling is the automated version of git fetch. It downloads a branch from a remote repository, then immediately merges it into the current branch. This is the Git equivalent of svn update.
